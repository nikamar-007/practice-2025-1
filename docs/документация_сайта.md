# Документация сайта "Поли"

## 1. Общая информация

Сайт `"Поли"` — это статичный адаптивный веб-сайт, выполненный полностью на `HTML` с использованием библиотеки утилитарных классов `TailwindCSS`. Сайт построен в виде набора отдельных HTML-страниц с общей структурой и стилем. Каждый компонент реализован средствами HTML и CSS-классов Tailwind без применения дополнительных скриптов.

---

## 2. Структура проекта

```
/
├── index.html         # Главная страница
├── about.html         # О проекте
├── journal.html       # Журнал разработки
├── team.html          # Участники команды
├── resources.html     # Внешние ресурсы    
└── images/           # Изображения и иллюстрации
```

Все страницы оформлены в едином стиле, используют одинаковые цветовые переменные и стили, подключаемые через встроенную конфигурацию `Tailwind` в теге `<script>`.

---

## 3. Общая структура HTML-страницы

Каждая страница начинается с шаблона:

```html
<!DOCTYPE html>
<html lang="ru">
  <head>
    <meta charset="UTF-8" />
    <title>Название</title>
    <script src="https://cdn.tailwindcss.com"></script>
    <link href="https://fonts.googleapis.com/..." rel="stylesheet">
    <style>/* Дополнительные классы и анимации */</style>
  </head>
  <body class="font-sans">
    <header>...</header>
    <main>...</main>
    <footer>...</footer>
  </body>
</html>
```

- Шрифт по умолчанию: `Inter`
- Цвета и отступы заданы через `tailwind.config`
- Весь макет адаптивен: сетки перестраиваются на разных экранах с помощью классов `md:*`

---

## 4. Шапка (header)

- Реализована как фиксированная (`sticky top-0 z-50`) полоса с белым фоном и тенями.
- Содержит логотип проекта (текст `Поли` с градиентом) и горизонтальное меню `<nav>` со ссылками на все страницы.
- Навигационные ссылки используют классы `hover:text-primary` и `font-semibold` для активной страницы.

```html
<header class="bg-white py-5 shadow-sm sticky top-0 z-50">
  <div class="max-w-6xl mx-auto px-4 flex justify-between items-center">
    <h1 class="text-2xl font-bold text-gradient">Поли</h1>
    <nav class="space-x-6 text-sm font-medium">...</nav>
  </div>
</header>
```

---

## 5. Главная страница (`index.html`)

### Hero-блок
- Использует `grid` с двумя колонками: текст + изображение.
- Заголовок (`h2`) имеет большие размеры и градиентную подсветку ключевого слова.
- Изображение персонажа "Поли" размещено в контейнере с псевдоэффектом свечения через `::after` и `drop-shadow`.
- Поверх изображения — сообщение в стиле "чат-баббла", спозиционированное через `absolute`.

### Преимущества
- Реализованы в виде карточек: `rounded-xl`, `shadow`, `border`, `p-6`
- Используются иконки SVG и тематические цвета (primary, secondary, accent)

### Блок заказчика
- Отдельный контейнер с названием и кнопкой перехода на сайт заказчика
- Используется `flex`, `justify-between` и `hover:bg-primary/20`

## 6. Страница "О проекте" (`about.html`)

### Вступительный блок
- Центрированный заголовок и подзаголовок + декоративная полоса `bg-gradient-to-r`

### Функциональность
- Блок с фоновым градиентом и белым текстом — выделяет описание функций для студентов и преподавателей
- Используются списки `<ul>` со стилем `flex items-start` и точками-иконками

### Преимущества
- 3 карточки (в сетке `md:grid-cols-3`) с описанием: "Экономия времени", "Уведомления", "Надежность"

### Работа бота (алгоритм)
- Левая часть: временная шкала (timeline), каждая точка оформлена через `::before`
- Правая часть: список особенностей (иконка + описание), стилизованный через `flex` и `space-y`

### Команды бота
- Каждая команда — карточка с рамкой, иконкой и монопрописью
- Используется `grid` из 2 колонок, при сужении экрана перестраивается

### Технологии
- Логотипы технологий оформлены в контейнерах с цветным фоном и текстом
- Приведена архитектура проекта (backend, API, БД) и блок статистики (кол-во строк кода, команд и пр.)

---

## 7. Страница "Журнал" (`journal.html`)

### Диаграмма Ганта
- Выполнена вручную с использованием `flex` и `position: absolute`
- Временные отрезки реализованы блоками `div.gantt-bar` с фоновыми цветами и закруглениями
- Сетка задается `repeating-linear-gradient` в `::before`

### Хронология
- Каждый этап — `timeline-item` с цветной точкой (`::before`) и блоком описания
- Используются отступы и `bg-slate-50` с `border` для выделения

### Статистика
- Три блока в сетке: количество команд, строк кода, активных пользователей
- Цветовая индикация (primary/secondary/accent)

---

## 8. Адаптивность

Сайт полностью адаптивен:

- Используются утилиты `md:`, `lg:`, `xl:` для перестроения сеток и изменения отступов
- Изображения и блоки с фиксированной шириной адаптируются через `max-w-full` и `mx-auto`
- На мобильных устройствах `grid-cols-2` заменяется на `grid-cols-1`
- Элементы, использующие `absolute`, переводятся в `relative` с переупорядочиванием на маленьких экранах

---

## 9. Пользовательские стили

Во всех HTML-файлах подключается `<style>` с переиспользуемыми классами:

- `.text-gradient` — градиентный текст с `background-clip`
- `.hover-scale` — масштабирование элемента при наведении
- `.floating-image` — анимация парения
- `.timeline-item::before` — точка в хронологии
- `.image-glow::after` — фоновое свечение

Анимации описываются через `@keyframes` прямо в `<style>`:
- `gentle-float` — плавное вертикальное движение
- `pulse-glow` — пульсирующее свечение фона

---

## 10. Заключение

Сайт "Поли" — это однородно оформленный, адаптивный, кроссбраузерный проект, реализованный **исключительно с помощью HTML и TailwindCSS**. Его страницы построены как набор переиспользуемых визуальных компонентов с минимальной логикой, что обеспечивает стабильную работу и лёгкую поддержку.
