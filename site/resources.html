Telegram разработка

  • node-telegram-bot-api

    Библиотека для работы с Telegram Bot API на Node.js

    Используется для обработки входящих сообщений и отправки ответов

Работа с данными

  • MySQL

    База данных MySQL для Node.js

    Хранение пользовательских данных и кэширование расписания

  • axios

    HTTP-клиент для Node.js и браузера

    Используется для запросов к API университета для получения расписания

Вспомогательные инструменты

  • node-cron

    Планировщик задач для Node.js

    Используется для автоматического обновления расписания по расписанию

  • chrono-node

    Парсер натуральных дат из текста

    Позволяет пользователям вводить даты в свободной форме ("завтра", "через 2 недели")

  • winston

    Система логирования для Node.js

    Запись ошибок и информационных сообщений для отладки
